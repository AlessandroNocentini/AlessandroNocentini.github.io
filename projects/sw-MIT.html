Project Description

Developed an Android application using the MIT App Inventor environment for the Application Logic, Google Sheets as Database and Google AppScript as Database Manager. The aim was to regulate the influx of people inside the stores during the Covid-19 pandemic period, avoiding also the creation of gatherings outside. This was made possible by allowing the user to book a free time slot in a desired supermarket, to create a ticket with a scannable QR Code. Provided the RASD, Design, Testing and Implementation documents of the project.
