Project Description

Built a functional Prusa i3 Hephestos 3D printer, mounting an Arduino Mega and a Ramps 1.4 boards. Assembled the hardware and set up the software, following the open-source project available in the RepRap website. Improved the machine by substituting the extruder component and adding a fan duct.
